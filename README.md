# Holansoft System 🚀

**Holansoft System** es una solución integral y moderna diseñada para la gestión administrativa de negocios, ofreciendo un control total sobre ventas, compras, inventarios y reportes, con un enfoque en la experiencia de usuario y el bilingüismo total.

---

## 🌟 Características Principales

### 📦 Gestión de Inventario Avanzada
- Control detallado de productos incluyendo **Marca, Tamaño y Tipo**.
- Alertas automáticas de stock bajo configurables.
- Categorización dinámica de productos.

### 💰 Ventas y Compras
- Flujo optimizado de transacciones.
- **Creación Rápida**: Agrega productos inexistentes al inventario directamente desde el módulo de compras.
- Historial detallado con desglose de productos, sub-totales e impuestos.

### 📊 Reportes e Inteligencia
- Reportes diarios y por lapsos de tiempo personalizables.
- Filtros avanzados por cliente, proveedor, producto o categoría.
- Visualización interactiva de detalles de transacciones.

### 🌎 Internacionalización (i18n)
- Soporte **100% bilingüe** (Español e Inglés) en toda la interfaz.
- Cambio de idioma instantáneo sin recarga de página.

### 🎨 UI/UX Premium
- Diseño moderno con **Modo Oscuro** y **Modo Claro**.
- Interfaz fluida basada en componentes interactivos y animaciones suaves.
- Totalmente responsivo para uso en diferentes dispositivos.

---

## 🛠️ Stack Tecnológico

- **Frontend**: React.js, Vite, Tailwind CSS, Lucide Icons.
- **Backend**: NestJS (Node.js), TypeORM, Prisma (para migraciones).
- **Base de Datos**: PostgreSQL.
- **Contenedores**: Docker & Docker Compose.

---

## 🚀 Instalación y Uso

### Requisitos previos
- Node.js (v18+)
- Docker & Docker Compose

### Pasos rápidos

1. **Clonar el repositorio**:
   ```bash
   git clone https://github.com/Di3g0-01/Holansoft_System.git
   cd Holansoft_System
   ```

2. **Levantar la infraestructura (Docker)**:
   ```bash
   docker-compose up -d
   ```

3. **Backend**:
   ```bash
   cd backend
   npm install
   npm run start:dev
   ```

4. **Frontend**:
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```
